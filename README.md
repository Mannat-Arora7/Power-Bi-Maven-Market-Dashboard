# Power-Bi-Maven-Market-Dashboard
Interactive Power BI dashboard built using the Maven Market dataset, featuring data modeling, DAX measures, and business insights through interactive visualizations.

<h1> Dashboard Preview </h1>
<br>

https://github.com/Mannat-Arora7/Power-Bi-Maven-Market-Dashboard/blob/main/Dashboard%20Preview.png

---

<h1> Project Overview </h1>
<br>
This project demonstrates the complete Power BI workflow, including:
<br>

<ul>
  <li> Data Cleaning & Transformation using Power Query </li>
  <li> Star Schema Data Modeling </li>
  <li> Calculated Columns & DAX Measures </li>
  <li> Interactive Dashboard Design </li>
  <li> Business KPI Analysis </li>
</ul>

---

<h1> Dashboard Features </h1>
<br>

<ul>
  <li> All Transactions by Month </li>
  <li> 60-Day Revenue by Product Brand </li>
  <li> Profit Margin by Product Brand and Name </li>
  <li> Quantity Sold by Last Month </li>
  <li> All Transactions by Month </li>
  <li> Quantity Returned by Month and Year </li>
  <li> Total Profit </li>
  <li> Total Revenue </li>
  <li> Total Transactions </li>
  <li> Total Returns </li>
  <li> Total Cost </li>
  <li> Country/Region and Product Brand </li>
  <li> Total Revenue by Year, Month and Day </li>
</ul>
